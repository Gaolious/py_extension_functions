Welcome to the Django Utils documentation!
===============================================

Django를 이용한 프로젝트에 공통적으로 쓰이는 module 모음 입니다.


시작하기
==========

pip를 이용한 설치::

 $ pip install py_extension_functions


설치 이후 Django Project의 settings ``INSTALLED_APPS`` 에 추가 해야 합니다. ::

  INSTALLED_APPS = (
      ...
      'gpp',
  )
